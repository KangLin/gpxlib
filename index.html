Purpose

A c++ library for parsing, browsing, changing and writing of GPX files. The specification of the GPX file format is listed here.

Documentation

The documentation for the gpx library can be found here.

Building

The gpxlib library depends on the STL and Expat. You will also need CMake (>=2.6.0) for building the library and the test programs.
