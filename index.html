Purpose

A c++ library for parsing, browsing, changing and writing of GPX files. The specification of the GPX file format is listed here.

Documentation

The documentation consists of a getting started manual, a description of the examples and a reference guide.

Building

The gpxlib library depends on the STL and Expat. You will also need CMake (>=2.8.0) for building the library and the test programs.

Package

No packages yet available. See github for the source.
