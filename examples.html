Examples using the gpxlib library

Introduction

In the example subdirectory there are a number of small examples. On the page these examples are described.

gpxls

The gpxls example is a little tool for showing a summary of the contents of a gpx file. The syntax of the tool is:

gpxls  FILE ...

The tool shows the attributes of the gpx root node and the number of waypoints, routes and tracks per file. It is an example how to use the parser.

gpxwpt

The gpxwpt example is a little tool for showing all the waypoint information in a gpx file. Its syntax is:

gpxwpt FILE ...

The tools shows for every waypoint per gpx file the latitude, longitude, elevation, time, name, comment, symbol and type. It is an example how to use the list of waypoints used in the library.
